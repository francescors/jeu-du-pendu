# Jeu du Pendu

Un jeu du pendu interactif développé en JavaScript avec Node.js, 
dont le dictionnaire est extrait du roman *Les Misérables* de Victor Hugo.

## Fonctionnalités

- 4 niveaux de difficulté : Facile, Moyen, Difficile, Impossible
- 10 tentatives avant la défaite
- Dictionnaire de mots issus des *Misérables*
- Interface interactive avec clavier alphabétique cliquable
- Affichage visuel progressif du pendu
- Page 404 personnalisée

## Technologies utilisées

- Node.js (serveur HTTP natif, sans framework)
- HTML / CSS / JavaScript

## Installation

1. Clonez le repository

git clone [URL_DU_REPO]

2. Lancez le serveur

node index.js

3. Ouvrez votre navigateur à l'adresse

http://localhost:8080

## Comment jouer

1. Choisissez un niveau de difficulté
2. Cliquez sur **JOUER**
3. Devinez les lettres en cliquant sur l'alphabet ou en tapant au clavier
4. Vous avez droit à 10 erreurs avant de perdre la partie
